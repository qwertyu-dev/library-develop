1.1 データ型と変数

概要

Pythonは動的型付け言語であり、変数の型を明示的に宣言する必要がありません。しかし、各変数には特定のデータ型が関連付けられています。主要なデータ型と変数の扱い方を学びましょう。

主要なデータ型

1. 数値型

整数 (int): 正数、負数、ゼロを表します。

浮動小数点数 (float): 小数点を含む数値を表します。

複素数 (complex): 実部と虚部を持つ数を表します。

# 数値型の例
int_num = 10
float_num = 3.14
complex_num = 1 + 2j

print(f"整数: {int_num}, 型: {type(int_num)}")
print(f"浮動小数点数: {float_num}, 型: {type(float_num)}")
print(f"複素数: {complex_num}, 型: {type(complex_num)}")

2. 文字列 (str)

テキストデータを表します。シングルクォート(') またはダブルクォート(") で囲みます。

# 文字列の例
single_quoted = 'Hello'
double_quoted = "World"
multi_line = """This is a
multi-line string"""

print(f"文字列1: {single_quoted}, 型: {type(single_quoted)}")
print(f"文字列2: {double_quoted}, 型: {type(double_quoted)}")
print(f"複数行文字列: \n{multi_line}")

3. リスト (list)

順序付けられた変更可能なシーケンスです。異なる型の要素を含むことができます。

# リストの例
mixed_list = [1, "two", 3.0, [4, 5]]
print(f"リスト: {mixed_list}, 型: {type(mixed_list)}")
print(f"リストの2番目の要素: {mixed_list[1]}")

4. タプル (tuple)

順序付けられた変更不可能なシーケンスです。

# タプルの例
my_tuple = (1, "two", 3.0)
print(f"タプル: {my_tuple}, 型: {type(my_tuple)}")

5. 辞書 (dict)

キーと値のペアのコレクションです。

# 辞書の例
person = {"name": "Alice", "age": 30, "city": "New York"}
print(f"辞書: {person}, 型: {type(person)}")
print(f"personの'name': {person['name']}")

6. セット (set)

重複のない要素の集合です。順序は保証されません。

# セットの例
my_set = {1, 2, 3, 1, 2}  # 重複は自動的に削除される
print(f"セット: {my_set}, 型: {type(my_set)}")

変数のスコープと名前空間

Pythonでは、変数のスコープは以下のように定義されます：

  • ローカルスコープ: 関数内で定義された変数
  • グローバルスコープ: モジュールレベルで定義された変数
  • ビルトインスコープ: Pythonの組み込み名前（関数や例外など）
# スコープの例
global_var = "I'm global"

def my_function():
    local_var = "I'm local"
    print(local_var)
    print(global_var)

my_function()
print(global_var)
# print(local_var)  # これはエラーになる

変数の命名規則

  • 文字、数字、アンダースコアを使用可能
  • 数字で始めることはできない
  • 大文字と小文字は区別される
  • 予約語（if, for, whileなど）は使用できない

型変換

Pythonでは、異なる型間で変換を行うことができます。

# 型変換の例
num_str = "10"
num_int = int(num_str)
num_float = float(num_str)

print(f"文字列: {num_str}, 型: {type(num_str)}")
print(f"整数に変換: {num_int}, 型: {type(num_int)}")
print(f"浮動小数点数に変換: {num_float}, 型: {type(num_float)}")
