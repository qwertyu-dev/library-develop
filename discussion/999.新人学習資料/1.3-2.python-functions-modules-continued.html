16. 次のコードの出力は何ですか？
def outer():
    x = "local"
    def inner():
        nonlocal x
        x = "nonlocal"
        print("inner:", x)
    inner()
    print("outer:", x)
outer()

a) inner: nonlocal
   outer: local
b) inner: nonlocal
   outer: nonlocal
c) inner: local
   outer: local
d) エラー
