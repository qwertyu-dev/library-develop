3. ファイル操作とI/O

3.1 ファイルの読み書き

Pythonでは、open()関数を使用してファイルを開き、読み書きを行います。

# ファイルの書き込み
with open('example.txt', 'w') as f:
    f.write('Hello, World!\n')
    f.write('This is a test file.\n')

# ファイルの読み込み
with open('example.txt', 'r') as f:
    content = f.read()
    print(content)

# 1行ずつ読み込み
with open('example.txt', 'r') as f:
    for line in f:
        print(line.strip())

3.2 CSVファイルの操作

CSVファイルの読み書きには、csvモジュールを使用します。

import csv

# CSVファイルの書き込み
data = [
    ['Name', 'Age', 'City'],
    ['Alice', '30', 'New York'],
    ['Bob', '25', 'London'],
    ['Charlie', '35', 'Tokyo']
]

with open('people.csv', 'w', newline='') as file:
    writer = csv.writer(file)
    writer.writerows(data)

# CSVファイルの読み込み
with open('people.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(', '.join(row))

3.3 JSON形式の扱い

JSONデータの読み書きには、jsonモジュールを使用します。

import json

# JSONデータの作成と書き込み
data = {
    'name': 'John Doe',
    'age': 30,
    'city': 'New York',
    'interests': ['programming', 'reading', 'traveling']
}

with open('person.json', 'w') as file:
    json.dump(data, file, indent=4)

# JSONファイルの読み込み
with open('person.json', 'r') as file:
    loaded_data = json.load(file)
    print(loaded_data)
